A Web3 endless runner game built on the Stacks blockchain. Earn STX and NFTs by playing!

## Quick Start
`ash
git clone https://github.com/Dark-Brain07/Stacks-Runner.git
cd Stacks-Runner/frontend && npm install && npm run dev
`

## Features
- Phaser 3 endless runner with procedural generation
- Stacks wallet integration (Leather / Xverse)
- On-chain score submission with STX rewards
- NFT minting for top players
- Progressive difficulty scaling
- Cyberpunk visual theme with particles
- PWA support for mobile install
- Docker deployment support
