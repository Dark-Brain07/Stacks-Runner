# Power-Up System
## Available
| Name | Duration | Effect |
|------|----------|--------|
| Shield | 5s | Blocks one hit |
| Coin Magnet | 8s | Auto-collects nearby coins |
| Double Points | 6s | 2x score multiplier |
| Time Warp | 4s | Slows obstacles |
| Shrink | 5s | Smaller hitbox |
| Phase Walk | 3s | Pass through obstacles |
## Implementation
Power-ups spawn randomly. PowerUpManager tracks states and expiration.
